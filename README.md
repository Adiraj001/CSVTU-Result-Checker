# 🎓 CSVTU Student Portal

A lightweight, user-friendly web portal that redirects students to the **official CSVTU result and admit card pages** based on roll number, semester, exam session, and exam type.

> ⚠️ This project does **not** host or modify results/admit cards.  
> It only redirects users to the **official CSVTU website**.

---

## ✨ Features

- 🔢 12-digit roll number validation
- 📄 View **Semester Results**
- 🪪 View **Admit Cards**
- 🎯 Supports **Regular, RTRV, and Backlog** exam types
- 📚 Semester & Exam Session selection
- 🔐 Redirects only to **official CSVTU portal**
- 📱 Fully responsive & modern UI
- ⚡ Fast, client-side only (no backend)

---

## 🔐 Important Notice (DigiVarsity Login Required)

> 🔑 **Users must log in to the DigiVarsity portal first** in order to access results or admit cards.  
> If you are not logged in, the CSVTU website may redirect you to the DigiVarsity login page.

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Modern responsive styling
- **JavaScript (Vanilla)** – Logic & redirection
- **CSVTU Official Portal (DigiVarsity)** – Data source (redirect only)
